Movie Recommender in Python

This movie recommender is programmed in python at backend .It makes use of matrix factorization algorithm alongside item-item collaborative filtering to recommend the most similar items ,here in this case movies to the user based on the most similar movies to the word entered in the search box.

Even if the movie entered in the search box is spelt wrongly, no worries pals b'coz my system makes use of another phenomenonal algorithm i.e Edit Distance to find the most similar word to the word entered and recommending based on that word.
